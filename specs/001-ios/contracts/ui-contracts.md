# UI 接口合同

本文档定义 Forest Focus Timer 应用的用户界面合同，确保一致的用户体验和可测试性。

## HomeView (主界面)

### 输入状态
- `durations`: [Int] = [15, 25, 45, 60]
- `selectedDuration`: Int (默认 25)

### 用户操作
| 元素 | 操作 | 预期结果 |
|------|------|----------|
| 时长选择器 | 选择时长 | `selectedDuration` 更新 |
| "开始植树"按钮 | 点击 | 导航到 FocusView，启动计时器 |

### 验收标准
- SC-003: 用户可以在 2 次点击内开始一个新的专注时段
- 界面文本不超过 500 字

---

## FocusView (专注中界面)

### 输入状态
- `remainingTime`: Int (秒)
- `currentStage`: Int (0-4)
- `isPaused`: Bool

### 用户操作
| 元素 | 操作 | 预期结果 |
|------|------|----------|
| 树木生长视图 | - | 显示当前阶段的 2D 插画 |
| 暂停按钮 | 点击 | 进入 PauseMenu，计时器暂停 |
| 放弃按钮 | 点击 | 显示摧毁动画，返回 HomeView |

### 视觉规范
- 树木阶段：5 个清晰的 2D 矢量插画阶段
- 倒计时显示：MM:SS 格式
- 动画：60 FPS 流畅过渡

---

## PauseMenu (暂停菜单)

### 输入状态
- `isPaused`: Bool

### 用户操作
| 元素 | 操作 | 预期结果 |
|------|------|----------|
| 恢复按钮 | 点击 | 退出暂停，继续计时 |
| 放弃按钮 | 点击 | 显示摧毁动画，记录未完成 |

---

## ForestView (森林视图)

### 输入状态
- `trees`: [Tree] (已完成的树木列表)
- `sortBy`: SortOption (.date, .duration)

### 用户操作
| 元素 | 操作 | 预期结果 |
|------|------|----------|
| 树木网格 | 点击单个树木 | 显示详情弹窗 (日期、时长) |
| 排序按钮 | 点击 | 切换排序方式 |
| 删除按钮 | 长按树木 | 显示删除选项 |

### 布局规范
- 网格布局：自适应列数 (根据屏幕宽度)
- 间距：16pt
- 每行树木数量：3-5 棵 (iPhone 尺寸适配)

---

## StatisticsView (统计面板)

### 输入状态
- `totalTrees`: Int
- `totalFocusTime`: TimeInterval
- `todayCount`: Int
- `currentStreak`: Int

### 显示指标
| 指标 | 格式 | 更新时机 |
|------|------|----------|
| 总树木数 | 数字 | 每次完成专注后 |
| 总专注时间 | "X 小时 Y 分钟" | 每次完成专注后 |
| 今日计数 | 数字 | 每日重置 (日历日) |
| 连胜记录 | "X 天" | 连续完成天数 |

### 验收标准
- SC-005: 统计数据在所有场景下始终准确

---

## 通用组件合同

### TreeStageView (树木阶段组件)

**输入**: `stage: Int` (0-4)
**输出**: 渲染对应阶段的 2D 矢量插画

### StatCard (统计卡片组件)

**输入**: `title: String`, `value: String`, `icon: String`
**输出**: 渲染统一风格的统计卡片

---

## 无障碍访问合同

### 所有界面元素必须满足

1. **VoiceOver 支持**
   - 所有可交互元素必须有 `accessibilityLabel`
   - 状态变化必须触发 `accessibilityAnnouncement`

2. **动态字体支持**
   - 使用 `Font.dynamicType()` 而非固定大小
   - 布局支持最大 200% 字体

3. **触控目标**
   - 最小触控区域：44x44pt

4. **对比度**
   - 文本与背景对比度：至少 4.5:1 (WCAG AA)

---

## 深色模式合同

### 颜色映射

| 浅色模式 | 深色模式 |
|----------|----------|
| 背景：#FFFFFF | 背景：#1C1C1E |
| 主色：#4CAF50 | 主色：#81C784 |
| 文本：#000000 | 文本：#FFFFFF |

### 资源适配
- 所有 2D 树木插画必须提供深色模式变体
- 使用 SF Symbols 的自动配色方案
